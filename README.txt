REQUIREMENTS

To build and run this application you need to have Maven and a JRE installed on your computer.

In order to build an executable file runnable from the desktop enviroment Maven needs for the JAVA_HOME environment variable to be set.

SETUP

Navigate to this folder in the console shell and run the command "mvn package". A directory, "target/", and an executable, "jumpy-birb.exe" should be created.

To run the application click on the "jumpy-birb" icon. Should this not appear run the command "java -jar target\jumpy-birb-0.0.1-SNAPSHOT.jar" in the console shell to start the program.
